|   中文   |         写法          |              示例               |
| :------: | :-------------------: | :-----------------------------: |
|    非    |         \bar          |            $\bar P$             |
|    并    |     \cup \bigcup      | $P \cup Q$    $\bigcup _{a}  P$ |
|    交    |     \cap \bigcap      |           $P \cap Q$            |
|  逻辑非  |         \lnot         |            $\lnot P$            |
|  逻辑并  |         \land         |           $P \land Q$           |
|  逻辑或  |         \lor          |           $P \lor Q$            |
|   蕴含   |          \to          |            $P \to Q$            |
|   任一   |        \forall        |           $\forall a$           |
|   存在   |        \exists        |           $\exists a$           |
|   空集   | \emptyset \varnothing |    $\emptyset  \varnothing$     |
|   属于   |          \in          |              $\in$              |
|  不属于  |        \not\in        |            $\not\in$            |
|  包含于  |        \subset        |            $\subset$            |
|   包含   |        \supset        |            $\supset$            |
| 真包含于 |       \subseteq       |           $\subseteq$           |
|  真包含  |       \supseteq       |           $\supseteq$           |
| 大于等于 |         \geq          |             $\geq$              |
| 小于等于 |         \leq          |             $\leq$              |
|  不等于  |         \neq          |             $\neq$              |
|  约等于  |        \approx        |            $\approx$            |
|  恒等于  |        \equiv         |            $\equiv$             |
|   加减   |          \pm          |              $\pm$              |
